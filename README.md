📚 Hub Leitura - Automação de Testes com Cypress

Este repositório foi desenvolvido como parte das atividades práticas do curso Engenharia de Qualidade de Software, da EBAC.

O objetivo do projeto é aplicar, na prática, conceitos de automação de testes para aplicações web utilizando o Cypress, além de desenvolver habilidades em versionamento de código com Git e GitHub.

🚀 Tecnologias utilizadas
Cypress
JavaScript
Node.js
Git
GitHub
📖 O que aprendi

Durante o desenvolvimento deste projeto, coloquei em prática conhecimentos sobre:

Configuração de um ambiente de automação com Cypress;
Estrutura de um projeto de testes automatizados;
Criação e organização de casos de teste em JavaScript;
Interação com elementos da página utilizando comandos do Cypress;
Automação do preenchimento de formulários;
Validação de comportamentos esperados da aplicação;
Execução e análise dos testes automatizados;
Versionamento do projeto utilizando Git;
Publicação e atualização do projeto no GitHub.

✅ Cenário automatizado

Foi desenvolvido um teste automatizado que:

Acessa a página de contato;
Preenche todos os campos do formulário;
Envia as informações;
Valida a mensagem de sucesso exibida pela aplicação.

🎯 Objetivo de aprendizado

Este projeto faz parte da minha jornada de transição para a área de Quality Assurance (QA). Meu objetivo é evoluir na criação de testes automatizados, aplicando boas práticas de qualidade de software e ampliando meu portfólio com projetos desenvolvidos durante o curso.
